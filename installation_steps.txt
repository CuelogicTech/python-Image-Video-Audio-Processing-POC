/*****************************************************************/
				*    Installation Process *
/****************************************************************/

=> Before starting any installation on Ubuntu run the below commands:

1. sudo apt-get update
2. sudo apt-get upgrade

=> Now run the below commands:

3. cd ~
4. mkdir python
5. cd /python

=> Now we'll create a virtual environment, for that need to install virtualenv package.

6. sudo pip install virtualenv  
	- For better understanding visit (http://docs.python-guide.org/en/latest/dev/virtualenvs/)

7. virtualenv avprocessing  
	- (avprocessing) Name for the virtual env, you can give the name whatever you want.

=> Now to use the created virtual env, it needs to be activated

8. source avprocessing/bin/activate 
(after executing this command you can see the name of your virtual env in the path, run the command 'pwd')
(NOTE: If you are done working in the virtual environment for the moment, you can deactivate it: run the command 'deactivate')


=> Now install all the dependencies required for the audio_video_image_processing library
=> Lets start with the opencv (it will take approx 1-2 hour for installation, depends upon your internet speed)

9. opencv Installation steps (Copy all & paste it in the terminal step-by-step, first 'a' then 'b' then 'c' & continues...)

	a.  sudo apt-get install build-essential libgtk2.0-dev libjpeg-dev libtiff4-dev libjasper-dev libopenexr-dev cmake python-dev 
		python-numpy python-tk libtbb-dev libeigen3-dev yasm libfaac-dev libopencore-amrnb-dev libopencore-amrwb-dev libtheora-dev libvorbis-dev libxvidcore-dev libx264-dev libqt4-dev libqt4-opengl-dev sphinx-common texlive-latex-extra libv4l-dev libdc1394-22-dev libavcodec-dev libavformat-dev libswscale-dev default-jdk ant libvtk5-qt4-dev

	b. cd ~
		wget http://sourceforge.net/projects/opencvlibrary/files/opencv-unix/2.4.9/opencv-2.4.9.zip
		unzip opencv-2.4.9.zip
		cd opencv-2.4.9

	c. mkdir build
		cd build
		cmake -D WITH_TBB=ON -D BUILD_NEW_PYTHON_SUPPORT=ON -D WITH_V4L=ON -D INSTALL_C_EXAMPLES=ON -D INSTALL_PYTHON_EXAMPLES=ON -D BUILD_EXAMPLES=ON -D WITH_QT=ON -D WITH_OPENGL=ON -D WITH_VTK=ON ..

	=> Now we need to compile & install the opencv
	d. make
		sudo make install

	e. sudo gedit /etc/ld.so.conf.d/opencv.conf 
		(it will open the opencv.conf file in edit mode. Copy '/usr/local/lib' & paste at the end of file & then save it & close the editor)

	f. sudo ldconfig

	g. sudo gedit /etc/bash.bashrc
		(it will open the bashrc file in edit mode. Copy below two lines & paste at the end of file & then save it & close the editor)
		PKG_CONFIG_PATH=$PKG_CONFIG_PATH:/usr/local/lib/pkgconfig
		export PKG_CONFIG_PATH

	h. Now you are done with the opencv library installation. Close the terminal & again run the below command:
		cd /python
		source avprocessing/bin/activate

10. ffmpeg installation 
	sudo apt-get install ffmpeg (wait untill it completes, it also take some time for installation)

11. moviepy installation (This library used to divide the video file in chunks, we need to send start & end position)
	sudo pip install moviepy

12. pyaudio installation (This library helps speech recognition library to extract text from audio)
	sudo pip install pyaudio

13. speechRecognition installation (As name specify it is basically used for speech-to-text conversion)
	sudo pip install SpeechRecognition

=> (Note: If you are getting an error while executing the step-12 & step-13. Run the below commands before executing the steps 12 & 13)
	- sudo apt-get install python-pyaudio
	- sudo apt-get install libjack-jackd2-dev portaudio19-dev

14. pillow installation (This library basically used for image processing & extracting text from image)
	sudo pip install Pillow

15. pytesseract installation (This library helps pillow for extracting text from image)
	sudo pip install pytesseract

16. pydub installation (This file help us to convert the audio file to .wav format)
		- before installing the pydub we need to install avlib, so execute the following command
		- sudo apt-get install libav-tools
		- pip install pydub
